# Roadmap

This roadmap is intentionally short. `v0.1.0` is a small initial release, not
the end state of the project.

## Near Term

- Stabilize the terminal-first launcher workflow across Local, Hybrid, and TUI
- Improve test coverage for the shared runtime and TUI integration points
- Tighten documentation around supported backends and model-management limits
- Reduce legacy naming and compatibility shims where they no longer help users

## Later

- Better release automation and packaging
- More robust TUI validation and platform coverage
- Stronger benchmark reporting with published reproducible runs
- Clearer migration story if prompts, tools, or CLI flags change

## Explicitly Not In `v0.1.0`

- Stable public plugin or extension API
- Hardened sandbox or security boundary
- Hosted service or GUI application
- Published benchmark claims
- Broad editor/IDE integration story
