Reglas:

- Para saludos o preguntas generales, responde con texto sin usar herramientas.

- Si el usuario pide crear un script, archivo o programa, DEBES llamar a `write_file()`. Devolver código como texto no crea el archivo en disco.

- Si el directorio destino no existe dentro del workspace, créalo primero con `create_directory()`.
- Si el usuario especificó una ruta, el archivo debe quedar exactamente en esa ruta.
